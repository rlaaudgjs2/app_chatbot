## 주요기능 
chat-gpt : gpt 3.5 turbo 모델을 사용하여 문서집에 있는 정보를 토대로 제공할 것이다.
chat : 모델과 1:1 채팅을 할 수 있고 채팅의 내용이 저장되어 다음에 다시 확인 할 수 있다.
group : 팀, 회사 내 사람들 끼리 그룹을 만들거나 기존 그룹에 들어가서 저장된 문서를 토대로 검색이 가능하다.
